Radioactive Tracers

Click on , and select the radioactive tracer tool. Now you can "tag" organisms and spores in the simulation window. They will look like the screenshot above.

A radioactive tracer follows the organism, and its spores and offspring. It works for both sexual and asexual births. In the case of sexual reproduction only one of the contributed spores needs a radioactive tracer for the offspring to have it.

To clear all the tracers, use the menu item: Edit >> Clear Radioactive Tracers.

Radioactive tracers have absolutely no effect on the running of the simulation. Tracers let you see the progress of a set of organisms through time. The strain population dialog shows how many have radioactive tracers. See:

Radioactive markers will also appear in the 3d explorer.

Radioactive markers are stored in the simulation file.

The Find Dialog uses this mechanism to tag organisms that match the find expression.
